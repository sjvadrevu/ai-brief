Anthropic cuts off third-party agent platforms from Claude subscriptions, forcing migration to API pricing

Anthropic blocked platforms like OpenClaw from accessing Claude through flat-rate subscriptions, requiring them to pay via API or usage add-ons. Agent workloads consume far more tokens than typical chat use and were straining capacity. Netflix released VOID, open-source physics-aware video object removal that rewrites scene dynamics. On research: GrandCode beat all human participants including grandmasters in three consecutive live Codeforces competitions.

What matters today

ANTHROPIC BLOCKS AGENT PLATFORMS

Anthropic cut off agent platforms including OpenClaw from Claude subscriptions, citing capacity strain from agent workloads that run around the clock and consume far more tokens than chatbot use.

  • Users must now pay via Claude's API or new usage add-ons; flat-rate subscription no longer covers agent tool access.
  • Anthropic had already imposed 5-hour session caps during peak hours before this broader restriction.
  • Timing is controversial: OpenAI is actively courting the same developer and agent-builder audience.
  • Reflects a structural tension: agent workloads are unbounded in consumption, making flat-rate pricing economically unviable at scale.
  • Agent platform operators face an immediate pricing model renegotiation with their own customers downstream.

NETFLIX VOID: PHYSICS VIDEO

Netflix released VOID, an open-source framework that removes objects from video while rewriting the physics consequences across the scene, not just painting over the gap.

  • Unlike typical inpainting tools, VOID simulates what physically happens in the scene after object removal.
  • Human evaluators preferred VOID results over Runway outputs two-thirds of the time.
  • Open-source release; applicable to VFX pipelines, content creation, and training data generation.

GRANDCODE: 311 VOTES

GrandCode is a multi-agent RL system using Agentic GRPO that took first place in three consecutive live Codeforces competitions, beating all human participants including legendary grandmasters.

  • Verified in real competitions against active human participants, not static benchmarks.
  • Agentic GRPO extends policy optimization to multi-stage agent rollouts, not just single-step outputs.
  • Top paper on HuggingFace this day with 311 votes.
