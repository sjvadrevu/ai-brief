1-bit LLMs hit browsers as Google, Anthropic, and China's humanoid factories all ship in a single day

PrismML's 1-bit Bonsai models bring sub-290MB inference to WebGPU browsers without Python or CUDA; Google launched Gemini 3.1 Flash TTS across 70 languages with 200+ audio control tags; Anthropic redesigned Claude Code desktop for parallel agent sessions; and Leju Robotics opened China's first automated humanoid factory at 10,000 units per year: while Hollywood's creative workforce reports a 50% income drop as studios accelerate AI adoption.

What matters today

PRISMML 1-BIT BROWSER LLMs

PrismML released 1-bit Bonsai, a series of commercially licensed 1-bit quantized LLMs that run directly in WebGPU-enabled browsers with no Python, CUDA, or specialized hardware required.

  • 1-bit Bonsai 1.7B compresses to 290MB (down from 3.4GB in FP16), enabling sub-1GB RAM execution via WebGPU.
  • Bonsai-8B achieves 14x compression vs FP16, with 1.06 intelligence density per GB compared to 0.10 for Qwen3-8B.
  • Inference speed: 40 tokens/sec on iPhone, 131 on M4 Pro, 368 on RTX 4090.
  • Energy consumption runs 4-5x lower than full-precision equivalents at equivalent output quality.
  • WebGPU compute shaders handle transformer operations, granting direct GPU hardware access from browser tabs.
  • Licensed under Apache 2.0 for commercial use.
  • Enables any web app to embed a private, offline inference layer without cloud API calls: applicable to contract review, local PII scrubbing, and offline enterprise workflows without infrastructure spend.

GOOGLE GEMINI 3.1 FLASH TTS

Google launched Gemini 3.1 Flash TTS on April 15 with support for 70+ languages, 200+ audio control tags, and native multi-speaker dialogue generation.

  • Scores Elo 1,211 on the Artificial Analysis TTS leaderboard.
  • 200+ audio tags control pace, tone, and vocal style at granular levels, exceeding ElevenLabs and OpenAI TTS style controls.
  • Native multi-speaker dialogue supports podcast-format and scripted conversations without post-processing.
  • All generated audio carries SynthID watermarking.
  • Available now via Gemini API, Vertex AI, Google AI Studio, and Google Vids in Workspace.
  • Embedding TTS natively into Google Vids puts voice directly in enterprise workflows without a separate API integration.
  • Ranked #6 on Product Hunt today with 110 votes.

ANTHROPIC CLAUDE CODE DESKTOP

Anthropic redesigned the Claude Code desktop app to support multiple concurrent agent sessions from a single window with persistent session management.

  • Session sidebar manages all active and recent sessions with filtering and grouping.
  • Drag-and-drop workspace layout allows custom arrangement of terminal, preview, diff viewer, and chat panels.
  • Side chat (Cmd+; / Ctrl+;) branches conversations without disrupting active tasks.
  • Revamped diff viewer handles large changesets; integrated terminal included.
  • SSH support extended to macOS, previously available on Linux only.
  • Three verbosity modes: Verbose, Normal, and Summary.
  • Available on Pro, Max, Team, Enterprise plans and via API.
  • Ranked #1 on Product Hunt today with 204 votes.

CHINA HUMANOID FACTORY

Leju Robotics and Dongfang Precision opened a factory in Foshan, Guangdong capable of producing 10,000 humanoid robots annually at one unit every 30 minutes.

  • Facility launched March 29; uses 24 precision assembly stages, 77 quality inspection checkpoints, and 41 simulated work-condition tests per unit.
  • Flexible multi-model production line requires no complete retooling to switch between robot variants.
  • 50% efficiency improvement cited over traditional assembly methods.
  • Customers include Nio, FAW Group (automotive), and Haier (appliances), targeting factory and assembly deployment.
  • At 10,000 units per year from a single facility, Chinese OEMs enter a cost-reduction curve not yet reached by US and European competitors including Figure, Apptronik, and 1X.

Research worth watching

  • GameWorld: Standardized evaluation for multimodal game agents

    Benchmark using semantic action spaces and state-verifiable metrics for evaluating multimodal LLM agents in video games. Verifiable metrics address the core problem with current agent evals: outputs are hard to confirm without ground truth. (82 votes)

  • OccuBench: AI agents on real-world professional tasks

    100 professional domains evaluated with Language World Models and controlled fault injection. The fault injection methodology provides a systematic approach to testing agent reliability before enterprise deployment. (45 votes)

  • Toward Autonomous Long-Horizon ML Research Engineering

    Hierarchical orchestration and durable state management for autonomous ML research. Structured coordination achieves superior benchmark performance over flat agent approaches. (28 votes)

  • The Blind Spot of Agent Safety

    Benign user instructions trigger harmful outcomes in computer-use agents with attack success rates exceeding 90%, even in safety-aligned models. Current safety training addresses malicious prompts, not emergent harmful behavior from ordinary instructions. (21 votes)
