Lazarus Group compromised OpenAI's macOS signing pipeline via Axios npm attack

North Korea's Lazarus Group accessed OpenAI's code-signing certificates for ChatGPT Desktop and Codex through a poisoned Axios npm package — forcing a certificate revocation deadline of May 8; Google shipped auto-spatialization and persistent sessions to Android XR on Samsung Galaxy headsets; and OpenBMB released VoxCPM2, a 2B-parameter open-source TTS model covering 30 languages with voice cloning under Apache-2.0.

What matters today

OPENAI NPM SUPPLY CHAIN

OpenAI confirmed that a malicious version of the Axios npm package (v1.14.1) executed inside its GitHub Actions workflow for macOS app-signing on March 31, 2026, exposing code-signing certificates for ChatGPT Desktop and Codex.

  • Google Threat Intelligence attributed the attack to UNC1069 (Lazarus Group), North Korea's primary state-sponsored threat actor.
  • OpenAI's GitHub Actions workflow used a floating tag instead of a pinned commit hash and lacked minimumReleaseAge configuration, the two controls that would have blocked the malicious package.
  • Attackers gained access to code-signing certificates for ChatGPT Desktop and Codex on macOS; no evidence of user data access, system compromise, or software alteration on iOS, Android, or Windows was found.
  • OpenAI is revoking its macOS certificate on May 8, 2026; after that date, macOS Gatekeeper will block older app versions from launching.
  • All macOS ChatGPT and Codex users must update before May 8 or their apps will stop launching.
  • The attack vector was the Axios npm HTTP client library, a widely-used package present in thousands of JavaScript projects, not the Axios news organization.

ANDROID XR AUTO-SPATIALIZATION

Google's April 2026 Android XR update for Samsung Galaxy XR headsets adds auto-spatialization, app pinning, session persistence, hand presence, and an expanded app ecosystem.

  • Auto-spatialization converts nearly any 2D app, website, image, or video into a 3D experience via a single tap in the Labs tab, requiring no developer changes.
  • App pinning lets users anchor applications to physical walls, where they persist in real-world space.
  • Session persistence automatically restores headset layouts including which apps were open and their exact positions.
  • Hand presence shows passthrough camera views of the user's hands for more natural interaction.
  • App ecosystem expanded to over 100 purpose-built immersive titles; single-eye tracking accessibility added for users with limited vision in one eye.

VOXCPM2 OPEN-SOURCE TTS

OpenBMB released VoxCPM2 on April 11, 2026, a 2-billion parameter tokenizer-free text-to-speech model under Apache-2.0 that runs on consumer hardware and supports 30 languages.

  • Trained on 2+ million hours of multilingual speech data, supporting 30 languages including English, Chinese, Spanish, French, German, Japanese, and nine Chinese dialects.
  • Maps text directly to speech in latent space without traditional tokenizers, generating 48kHz studio-quality audio.
  • Three operating modes: voice design from text descriptions, controllable voice cloning with style guidance, and ultimate cloning from reference audio.
  • Requires approximately 8GB VRAM for inference; achieves approximately 0.13 real-time factor on consumer GPUs.
  • Apache-2.0 license removes the commercial restrictions that limit ElevenLabs API alternatives.
