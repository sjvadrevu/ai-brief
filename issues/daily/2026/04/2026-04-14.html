April 14, 2026 Tuesday edition

Figma opens its canvas to agents as AWS ships AgentCore and LG releases a 33B open VLM

Figma gave AI agents write access to its live canvas via a native MCP server; AWS brought Amazon Bedrock AgentCore and a Spring AI SDK to general availability for Java developers; the UK Department for Transport published results showing Gemini cutting public consultation processing from months to hours; and LG AI Research released EXAONE 4.5, a 33-billion-parameter open-weight vision-language model that outscores GPT-5-mini and Claude 4.5 Sonnet on STEM benchmarks.

What matters today

FIGMA CANVAS AGENTS

Figma released a native MCP server giving AI agents full write access to the Figma canvas, enabling creation and editing of components using a team's live design system.

  • MCP server is free and open-source; integrates with Claude, ChatGPT, and Cursor.
  • Agents have full create/edit access to components, not just file inspection.
  • A Skills framework lets teams encode design conventions in markdown files; well-organized design systems directly determine agent output quality.
  • Nine community-built skills launched at release, covering token sync and accessible annotation generation.
  • Figma for Agents reached #2 on Product Hunt with 177 votes.
  • The bottleneck shifts from agent capability to design system organization: messy token libraries produce messy outputs; rigorous component libraries produce on-brand components on the first prompt.

AWS AGENTCORE GA

Amazon Bedrock AgentCore reached general availability alongside a Spring AI SDK, making production agent deployment native for Java/Spring developers.

  • AgentCore GA adds VPC and AWS PrivateLink support for enterprise network isolation.
  • CloudFormation integration enables infrastructure-as-code deployments.
  • Quality evaluations with policy controls added for enterprise agent governance.
  • Spring AI SDK now GA; Java/Spring teams can build agentic apps without leaving their stack.
  • SDK downloaded over one million times across industries prior to GA.
  • Java is the dominant language in enterprise back-office systems; native Spring support removes language-switching friction for the teams AI agents are designed to interact with.

UK TRANSPORT GEMINI DEPLOYMENT

The UK Department for Transport deployed a Consultation Analysis Tool built on Google Cloud Vertex AI with Gemini, cutting processing of 100,000+ public responses from months to hours.

  • Built on Vertex AI with Gemini models; deployed via Cloud Run, Cloud CDN, and Firestore.
  • Processed over 100,000 free-text responses for the Integrated National Transport Strategy.
  • Achieved up to 90% accuracy across theme identification and categorization metrics.
  • Expected to save the Department for Transport up to £4 million annually.
  • Developed in partnership with Google Cloud and the Alan Turing Institute.
  • Human-in-the-loop review retained for final accuracy and bias validation.

LG EXAONE 4.5 VLM

LG AI Research released EXAONE 4.5, a 33-billion-parameter open-weight vision-language model targeting enterprise document understanding with a 262K token context window.

  • 33B parameter VLM with 262K token context window; integrates visual encoder into EXAONE 4.0 architecture.
  • STEM benchmark score 77.3 exceeds GPT-5-mini (73.5) and Claude 4.5 Sonnet (74.6).
  • LiveCodeBench v6 score 81.4; 92.9% on AIME 2025 math competition.
  • Supports six languages: Korean, English, Spanish, German, Japanese, and Vietnamese.
  • Hybrid attention architecture: alternating sliding-window and global attention across 64 layers.
  • Non-commercial license restricts use to research and academic contexts only.

SOFTR AI CO-BUILDER

Softr launched an AI Co-Builder that generates fully functional business applications including database, UI, authentication, permissions, and third-party integrations from a single natural language prompt.

  • Single prompt generates a complete app: database, UI, auth, permissions, and integrations.
  • Assembles pre-built, tested components rather than generating raw code, producing day-one-functional results.
  • Targets non-technical business teams replacing manual spreadsheet and workflow processes.
  • Covers internal tools, customer portals, partner platforms, and business automation use cases.
  • Softr AI Co-Builder ranked #1 on Product Hunt with 184 votes.

Research worth watching

  • QuanBench+

    Benchmark for LLM quantum code generation across multiple frameworks using functional testing and repair-based feedback. LLMs show progress but remain dependent on framework-specific knowledge.

  • MEDS: Memory-Enhanced Dynamic Reward Shaping

    RL framework that penalizes recurrent error patterns by clustering historical behavioral signals, improving sampling diversity in LLM post-training.

  • Attention Sink in Transformers: A Survey

    Comprehensive survey on the attention sink phenomenon where models over-attend to uninformative tokens, covering interpretability implications and mitigation strategies.

  • EXAONE 4.5 Technical Report

    Full technical writeup for LG's 33B open-weight VLM. Architecture details, benchmark methodology, and training approach for the model covered in today's main story.
