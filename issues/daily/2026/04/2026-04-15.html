Anthropic buys Bun and Claude Code crosses $1B as Google ships Gemini Robotics-ER 1.6

Anthropic acquired the Bun JavaScript runtime to power Claude Code as the coding agent crossed $1 billion in annualized revenue, while Google DeepMind released Gemini Robotics-ER 1.6 with a fourfold jump in instrument-reading accuracy, and OpenAI launched GPT-5.4-Cyber behind a tiered identity-verification program for security professionals.

What matters today

ANTHROPIC ACQUIRES BUN

Anthropic acquired Bun, the all-in-one JavaScript runtime built by Jarred Sumner, to anchor the coding infrastructure behind Claude Code as the agent hit $1 billion in annualized revenue.

  • Bun combines runtime, package manager, bundler, and test runner in a single tool, replacing the need to manage separate dependencies.
  • Claude Code crossed $1B annualized run-rate in November 2025, six months after its May 2025 public launch.
  • Bun has 7+ million monthly downloads and 82,000+ GitHub stars at the time of acquisition.
  • Enterprise users of Bun include Netflix, Spotify, Midjourney, Lovable, KPMG, L'Oreal, and Salesforce.
  • Bun will become the runtime foundation for Claude Code, the Claude Agent SDK, and future AI coding products.
  • Bun remains open source under the MIT license post-acquisition.

GEMINI ROBOTICS-ER 1.6

Google DeepMind released Gemini Robotics-ER 1.6, a robotics-focused embodied reasoning model with large gains in visual and spatial accuracy over ER 1.5, now available via Gemini API and Google AI Studio.

  • Instrument reading accuracy improved from 23% (ER 1.5) to 93% (ER 1.6) using agentic vision.
  • Pointing accuracy in object identification and counting tasks: 80% success rate.
  • Multi-view detection in occluded environments: 84% success rate.
  • Built with Boston Dynamics for autonomous reading of industrial gauges and sight glasses.
  • 10% improvement over Gemini 3.0 Flash on video-based injury risk detection.

OPENAI GPT-5.4-CYBER

OpenAI announced Trusted Access for Cyber, an identity-verification program granting vetted cybersecurity professionals access to GPT-5.4-Cyber, a version of GPT-5.4 fine-tuned with lower refusal thresholds for defensive security work.

  • GPT-5.4-Cyber allows capabilities restricted in standard models, including binary reverse engineering and vulnerability research.
  • Access tiers are determined by verified identity level, not organization type.
  • Highest-tier users can request access to advanced defensive workflows.
  • OpenAI is not currently offering GPT-5.4-Cyber to U.S. government agencies; discussions are ongoing.
  • The program shifts from restricting model capabilities to controlling who can access them.

Research worth watching

  • ClawGUI: A Unified Framework for Training, Evaluating, and Deploying GUI Agents

    Open-source framework addressing environment instability and persistent memory management in GUI agent deployment. Standardizes RL training and cross-platform evaluation for production agent systems.

  • KnowRL: Boosting LLM Reasoning via Reinforcement Learning with Minimal-Sufficient Knowledge Guidance

    Knowledge-guided RL framework that optimizes compact, interaction-aware guidance subsets to solve reward sparsity and knowledge pruning in LLM reasoning chains.

  • Rethinking On-Policy Distillation of Large Language Models

    Analyzes when on-policy distillation works and fails. Success requires compatible thinking patterns between teacher and student; teachers must provide capabilities absent from the student model.

  • SPPO: Sequence-Level PPO for Long-Horizon Reasoning Tasks

    Reformulates long-chain-of-thought reasoning as a contextual bandit problem with decoupled value functions. Improves temporal credit assignment and sample efficiency for extended reasoning chains.
