Anthropic ships dual-model advisor API as Amazon discloses $15 billion AWS AI run rate

Anthropic opened its Opus advisor API to developers, pairing high-capability reasoning with cheaper Sonnet or Haiku executors; Amazon's Andy Jassy put a $15B annual run rate on AWS AI and defended a $200B capex commitment; Perplexity hit $450M ARR with full bank account integration via Plaid; AWS launched a governed agent catalog inside Bedrock AgentCore; Oxford published an AI tool predicting heart failure five years early from CT scans; and France set a government-wide deadline to exit Windows for Linux.

What matters today

ANTHROPIC OPUS ADVISOR API

Anthropic shipped the advisor tool (beta header advisor-tool-2026-03-01) for the Claude API, enabling a dual-model architecture where Opus acts as an on-demand consultant to a Sonnet or Haiku executor.

  • Sonnet + Opus advisor delivers +2.7 percentage points on SWE-bench at -11.9% cost vs. Sonnet alone.
  • Haiku + Opus advisor reaches 41.2% on BrowseComp, over 2x Haiku's solo 19.7%, at 85% lower cost than Sonnet alone.
  • Advisor responses average 400-700 tokens and integrate with web search, code execution, and other tools in the same call.
  • Token billing is separated at each model's rate; max_uses parameter caps advisor invocations per run.
  • Full API docs are available on the Claude platform.

AMAZON AWS AI INFRASTRUCTURE

Andy Jassy's 2026 shareholder letter disclosed that AWS AI has reached a $15 billion annual run rate, with Amazon's custom chip business generating over $20 billion annually.

  • Trainium2 is fully sold out; Trainium3 is nearly fully subscribed.
  • 2026 capex commitment of approximately $200 billion is primarily for AI infrastructure.
  • Amazon's free cash flow fell from $38 billion to $11 billion as capital spending rose by $50.7 billion.
  • Jassy's framing in GeekWire: "We're not investing on a hunch."

PERPLEXITY BANK INTEGRATION

Perplexity expanded its Plaid integration beyond investment accounts to include checking, savings, credit cards, loans, and mortgages.

  • Plaid powers nearly one million daily account connections and reaches more than half of American bank account holders.
  • Perplexity's Computer agent drove a 50% monthly revenue jump, pushing the company to $450M annualized run rate as of March.
  • Feature is live on desktop for US and Canada users; supports queries on spending by category, net worth, and debt payoff timelines.

AWS BEDROCK AGENT REGISTRY

AWS Agent Registry, part of Amazon Bedrock AgentCore, launched in preview on April 9 as a private, searchable catalog for agents, tools, skills, MCP servers, and custom enterprise resources.

  • Semantic search lets developers query in natural language: searching "payment processing" surfaces tools tagged as "billing" or "invoicing."
  • Approval workflows and CloudTrail audit trails address enterprise governance requirements.
  • Accessible via Console, CLI/SDK, or as an MCP server queryable from IDEs.
  • Available in five regions; free during preview period.

FRANCE LINUX TRANSITION

France announced a government-wide transition from Windows to Linux, with every ministry required to submit implementation plans by autumn 2026, led by DINUM under a digital sovereignty mandate.

  • Scope extends beyond desktops: collaboration tools, antivirus, AI platforms, databases, and network equipment are all in scope.
  • 80,000 National Health Insurance Fund employees already migrated from Microsoft Teams, Zoom, and Dropbox to open-source alternatives.
  • Coverage: TechCrunch and Tom's Hardware.

OXFORD HEART FAILURE AI

A team at the University of Oxford, led by Professor Charalambos Antoniades, published an AI tool that detects heart failure risk from routine CT scans by identifying microscopic textural changes in cardiac fat tissue.

  • Tested on 70,000+ patients across nine NHS Trusts over a decade; algorithm achieved 86% accuracy.
  • Patients in the highest-risk group were 20 times more likely to develop heart failure than those in the lowest group.
  • Team is seeking NHS regulatory approval and plans to expand the tool to any chest CT, including lung imaging.
