AI Brief
April 17, 2026 Thursday edition

Anthropic, OpenAI, Google, and Canva all release agentic AI tools in a 24-hour window

Claude Opus 4.7 posts a 10-point SWE-Bench jump as OpenAI expands Codex into a desktop automation agent; Google embeds AI Mode directly into Chrome with side-by-side browsing; Canva launches a full agentic design platform with workflow connectors; Alibaba open-sources a sparse MoE model that activates just 3B of 35B parameters.

What matters today

ANTHROPIC CLAUDE OPUS 4.7

Anthropic released Claude Opus 4.7, scoring 64.3% on SWE-Bench Pro, nearly 10 points above Opus 4.6.

  • 64.3% SWE-Bench Pro score, up approximately 10 points from Opus 4.6.
  • New xhigh effort level sits above high for tasks where latency tolerance is high.
  • Self-verification capability lets the model generate a checking strategy before finalizing outputs.
  • Vision resolution raised to 2,576px on the long edge.
  • Pricing unchanged at $5 input / $25 output per million tokens.
  • Available on API, Amazon Bedrock, Google Cloud Vertex AI, and Microsoft Foundry.

OPENAI CODEX DESKTOP AGENT

OpenAI updated Codex on April 16, adding multi-agent workflows, desktop computer control, and 90+ plugin integrations.

  • Multi-agent workflows run writing, debugging, and testing agents simultaneously.
  • Desktop control allows Codex to view applications and perform clicks and keystrokes across macOS and Windows.
  • 90+ new plugins including Atlassian Rovo, CircleCI, GitLab Issues, and the Microsoft Suite.
  • Persistent memory supports long-horizon cross-session projects.
  • In-app browser with image generation capabilities included.

GOOGLE AI MODE IN CHROME

Google shipped AI Mode directly into Chrome starting April 16 in the US, across desktop, Android, and iOS.

  • Side-by-side browsing: web page and AI Mode panel stay simultaneously visible rather than replacing each other.
  • Plus menu allows adding any open tab, image, or file as query context.
  • New Tab Page prompt appears inline on desktop with no redirect to a separate page.
  • Mobile gets a fullscreen interface with camera, gallery, and file access.
  • Supports Gemini 3 model switching, Canvas, Deep Search, and image generation from within the browser.

CANVA AI 2.0 AGENTIC DESIGN

Canva launched AI 2.0 on April 16 as a research preview for the first 1 million users, enabling text-to-design with persistent memory and workflow connectors.

  • Text prompts generate fully editable, layered multi-format campaigns from a single prompt.
  • Persistent memory retains user preferences across sessions.
  • Workflow connectors link to Slack, Gmail, Google Drive, Calendar, and Zoom.
  • Internal models Proteus, Lucid Origin, and I2V run 7x faster and 30x cheaper than comparable frontier alternatives.
  • Research preview open to the first 1 million users.

ALIBABA QWEN3 SPARSE MOE

Alibaba released Qwen3.6-35B-A3B, a sparse MoE vision-language model with 35B total parameters but only 3B active at inference time.

  • 35B total parameters with only 3B activated at inference, delivering major compute reduction.
  • Sparse MoE architecture maintains multimodal vision-language performance.
  • Qwen3-Next 80B-A3B uses 512 routed experts plus one shared expert, activating 10 per token.
  • 260K+ token context length supported via Gated Delta Networks.
  • Open-source and deployable on NVIDIA NIM, SGLang, or vLLM.

Research worth watching

  • HY-World 2.0

    Multi-modal world model generating high-fidelity 3D Gaussian Splatting scenes from diverse inputs, with panorama generation, trajectory planning, and an interactive 3D rendering platform.

  • DR3-Eval

    Benchmark for deep research agents on multimodal, multi-file report generation, evaluating factual accuracy, citation coverage, and hallucination control in simulated web environments.

  • HiVLA

    Hierarchical vision-language-action framework decoupling semantic planning from motor control, using a diffusion transformer with cascaded cross-attention for zero-shot robotic manipulation.

  • ASGuard

    Uses mechanistic circuit analysis to identify attention heads vulnerable to jailbreaking and recalibrates them through targeted fine-tuning, without full model retraining.
